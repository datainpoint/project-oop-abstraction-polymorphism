## TL; DR

在這個專案中，我們打算使用 Python 定義兩個類別：分別是 `SimpleCalculator` 與 `AdvancedCalculator`；其中 `SimpleCalculator` 類別具有四個成員方法以及一個成員屬性；`AdvancedCalculator` 類別繼承了 `SimpleCalculator` 類別並新增一個方法。透過這個專案，我們能夠暸解何謂物件導向的第二支柱：繼承。
